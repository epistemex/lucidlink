lucidlinks
==========

A small utility to open lucid:// links on KDE Plasma / Linux for Dolphin.

Context:
https://www.lucidlink.com/classic

Install
-------

From the root-folder:

    npm i -g .

Usage:
------

In the terminal:

    lucidlink lucid://.......
